# Data Types

## Boolean
* Can only be `true` or `false`
```javascript
var bool = true
var bool = false
```

## Number
* represents a number: (1, 1.4, 0, etc.)
```javascript
var number = 5
```

## String 
* the one with the quotes

## Null 
* represents no value or nothing

## undefined 
* same as null

## Objects
* anything else 
```javascript
var object = {
    key: "value"
}
```
